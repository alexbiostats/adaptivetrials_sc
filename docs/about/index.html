About This Website

This is an R Markdown website, created by Alex Kaizer for a short course on adaptive and Bayesian clinical trial designs for the 2024 WNAR/IMS/Graybill Conference hosted at Colorado State University.

In the short course, we’ll actively use several packages which are listed below. Code to install the packages follows, and version information about each of these packages and their dependencies is at the bottom of this page.

For step-by-step instructions on installing brms and Stan, the following learnB4SS provides nice steps on their GitHub website.

Installing packages from CRAN

# main packages
install.packages(c("brms","bayestestR","rpact","carat","kableExtra","blindrecalc","gsDesign"))
